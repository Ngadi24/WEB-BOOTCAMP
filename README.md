# WEB-BOOTCAMP

## Description
This repository contains a beginner-friendly JavaScript project created as part of a web development assignment. The project demonstrates the use of HTML elements, JavaScript DOM manipulation, and class-based element selection.

## Project Objective
The main objective of this project is to practice:
- Creating multiple HTML elements with the same class name
- Accessing elements using `document.getElementsByClassName`
- Logging selected elements to the browser console

## Technologies Used
- HTML
- JavaScript

## How It Works
- Three paragraph (`<p>`) elements are created with the same class name.
- JavaScript is used to select these elements using the class name.
- The selected elements are logged to the console for verification.

## How to Run the Project
1. Clone or download the repository.
2. Open the project folder.
3. Open the HTML file in any web browser.
4. Open the browser console to view the logged output.

## Learning Outcome
This project helped reinforce understanding of:
- DOM manipulation
- JavaScript selectors
- Basic front-end development workflow
